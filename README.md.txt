# Sistema de Cadastro de Usuários

Projeto desenvolvido em Python com o objetivo de praticar lógica de programação, CRUD e manipulação de arquivos.

## Funcionalidades
- Cadastrar usuários
- Listar usuários
- Editar usuários
- Deletar usuários
- Salvamento automático em arquivo JSON

## Tecnologias utilizadas
- Python
- JSON
- HTML
- CSS

## Como executar
1. Clone o repositório
2. Execute o arquivo:

```bash
python main.py
