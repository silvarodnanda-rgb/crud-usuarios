# Sistema de Cadastro de Usuários

Projeto desenvolvido em Python com o objetivo de praticar lógica de programação, CRUD e Banco de Dados

## Funcionalidades

- Login 
- Cadastrar usuários
- Listar usuários
- Editar usuários
- Deletar usuários
- Salvamento automático em tabela SQL
- Logout

## Tecnologias utilizadas
- Python
- HTML
- CSS
- SQLite 
- Flask 
- Werkzeug

## Como executar
1. Clone o repositório
2. Instale o sqlalchemy e o werkzeug digitando no terminal do Git Bash: pip install flask flask_sqlalchemy werkzeug 
3. Execute o arquivo:

```bash
python main.py
